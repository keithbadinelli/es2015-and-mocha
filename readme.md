# Introduction to ES2015 and Javascript Testing with Mocha and Chai

## View Presentation
```bash
  $ open index.html
```
(or manually open index.html in a browser)

## Setup Test Project
```bash
  $ npm i
```

## Run Test Cases
```bash
  $ npm run test
```
